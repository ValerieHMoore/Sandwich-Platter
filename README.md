Welcome to Sandwich Platter!

Description:
Users can sign up or login to see close to 200 sandwich recipes.
You can only Create, Update or Delete your own sandwiches.
This web app was built in Sinatra using the Corneal Gem.

Install Instructions:
To install this app, fork and clone from GitHub.
CD into the app folder and then Bundle Install all gems.
Type shotgun in your terminal and navigate to localhost:9393 in your browser.
Bon Appetit!
